# Calculadora de Impacto Ambiental para Viagens

Esta é uma calculadora simples para estimar o impacto ambiental de viagens, focando nas emissões de CO2.

## Funcionalidades

- Calcula emissões de CO2 baseadas na distância e no meio de transporte.
- Suporta diferentes modos: carro, avião, trem, ônibus.

## Como Usar

1. Execute o script `main.py`.
2. Insira a distância da viagem em quilômetros.
3. Escolha o meio de transporte.
4. Veja o resultado das emissões estimadas.

## Instalação

Certifique-se de ter Python 3 instalado.

Clone o repositório:
```
git clone https://github.com/seu-usuario/calculadora-impacto-ambiental-viagens.git
cd calculadora-impacto-ambiental-viagens
```

Execute:
```
python main.py
```
